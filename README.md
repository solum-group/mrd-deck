# MRD 작성 가이드 — SOLUM PM Group

MRD(Market Requirements Document) 팀 온보딩 워크샵 발표자료입니다.

## 보기

배포 후 README의 링크를 업데이트하세요.

## 조작 방법

| 입력 | 동작 |
|---|---|
| `→` / `Space` | 다음 슬라이드 |
| `←` | 이전 슬라이드 |
| `Home` | 첫 슬라이드 |
| `End` | 마지막 슬라이드 |
| 스와이프 | 모바일 좌우 슬라이드 |

## 구성 (27장)

- Section 00: Stakeholder Matrix — RACI
- Section 01: MRD란?
- Section 02: Executive Summary + Success Metrics
- Section 03: Market Opportunity
- Section 04: Customer & Problem
- Section 05: Solution Overview + Feature Decomposition
- Section 06: Market Requirements (MoSCoW + PRD 전환 포인트)
- Section 07: Competitive Analysis
- Section 08: GTM Considerations
- Section 09: Constraints & Assumptions
- Section 10: MRD → PRD 핸드오프 가이드
